# 00. 프로젝트 개요 (Project Overview)

## 1. 프로젝트명

- **Kirakira**

## 2. 프로젝트 목표

- 건담 시리즈에 등장하는 다채로운 시각 효과(예: 미노프스키 입자, GN 입자, 뉴타입 섬광 등)를 Three.js를 활용하여 인터랙티브한 3D 웹 페이지로 구현.
- 사용자에게 각 효과에 대한 정보와 함께 시각적인 즐거움을 제공하는 것을 목표로 함.

## 3. 핵심 기능

- **3D 효과 뷰어**: 선택된 건담 효과를 3D 공간에서 렌더링하고 사용자가 직접 조작할 수 있는 기능.
- **효과 라이브러리**: 다양한 효과 목록을 탐색하고 선택할 수 있는 인터페이스.
- **상세 정보 패널**: 각 효과의 설정, 배경, 관련 기체 등 상세 정보를 제공.
- **인터랙티브 컨트롤**: 사용자가 실시간으로 효과의 파라미터(색상, 밀도, 속도 등)를 변경하며 변화를 관찰할 수 있는 기능.

## 4. 기술 스택

- **Build Tool**: Webpack
- **Framework**: Vue.js 3 (Composition API)
- **3D Library**: Three.js
- **State Management**: Pinia
- **Routing**: Vue Router
- **Styling**: CSS (PostCSS) 또는 SCSS

## 5. 디자인 컨셉

- **Sleek, Futuristic, Clean**: 세련되고 미래지향적이며, 깔끔한 디자인.
- 어두운 계열의 배경색과 네온 색상의 포인트 컬러를 사용하여 SF적인 분위기를 강조.
- 직관적인 UI와 미니멀한 아이콘을 통해 사용성을 높임.
