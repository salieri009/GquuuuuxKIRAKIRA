# RES-003: Effect Catalog & Implementation Guide

## Overview
Complete catalog of all visual effects in the Kirakira project, organized by Universe timeline with implementation-ready specifications.

> **Why this document exists:** Serves as a quick reference for developers, consolidating effect parameters from detailed research into a single implementation checklist. Maps effect IDs to colors, timelines, and parameter schemas for rapid development.

## Universal Century Effects

### Effect: Minovsky Particles (`minovsky-particles`)

**Description:** Core technology of UC timeline. Blue particles that interfere with electromagnetic signals, forcing visual-range combat.

**Visual Style:**
- Subtle, atmospheric
- Low visibility particles
- Interference/static effects
- Fog-like density

**Colors:**
| Token | Hex | Usage |
|-------|-----|-------|
| `--minovsky-primary` | `#00AAFF` | Base particle |
| `--minovsky-interference` | `#FF6666` | Sensor static |
| `--minovsky-fog` | `rgba(0,170,255,0.08)` | Atmosphere |

**Parameters:**
```json
{
  "density": { "type": "slider", "value": 0.3, "min": 0.1, "max": 1.0 },
  "turbulence": { "type": "slider", "value": 0.5, "min": 0.1, "max": 2.0 },
  "color": { "type": "color", "value": "#00AAFF" },
  "opacity": { "type": "slider", "value": 0.7, "min": 0.1, "max": 1.0 }
}
```
